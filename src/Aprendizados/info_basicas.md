# Tipos de Operadores e Outros :
| Símbolo da Operação | Função/Significado |
| -------- | -------- |
|         `+`         | Somar |
|         `-`         | Subtrair |
|         `*`         | Multiplicar |
|         `/`         | Dividir |
|     `#` ou `"""`    | Adicionar um Comentário |
|         `=`         | Atribuição |
|         `==`        | `var` igual a `var`  |
|         `!=`        | `var` diferente de `var`  |
|         `>`         | `var` maior do que `var`  |
|         `<`         | `var` menor do que `var`  |
|         `>=`        | `var` maior ou igual a `var`  |
|         `<=`        | `var` menor ou igual a `var`  |

# Tipos de Variáveis e Outros :
|   Tipo de Variável  | Significado |
| -------- | -------- |
|         `int`       | Tipo Inteiro |
|         `str`       | Tipo String |
|         `float`     | Tipo String |
|         `*`         | Multiplicar |
|         `/`         | Dividir |

# Palavras-Chave :
| Palavra | Significado |
| --------- | --------- |
| `var` | Variável |
| `class` | Criação de uma Classe |
| `print` | Executar algo na tela |
| `%s` | Usado para Variáveis `String` |
| `%d` | Usado para Variáveis `Int` |
| `%(var)` | Usado para pegar o valor de uma variável e colocar-la num `print()` |

# Instruções Básicas :
```python
# === Executar um Print === #
print("Isto Aqui !") #Programa escreve o que está dentro de ""
```
